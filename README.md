# tools
Jsを勉強して作ったツール&見つけたツールを載せます。(勉強中だから載せれるの1年後とかになりそう)

## bio.js
```js
javascript:{var x=document.createElement("script");x.src="//ptbnth.github.io/tools/bio.js";document.body.appendChild(x)}
```
Scratchの私についてのところに「Last Login」と打ち実行すると、現在時間が右側に表示される(生存確認)
一回実行したら、タブを移動するまで実行しなくて大丈夫
